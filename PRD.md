## 2. 目标受众

- **主要**：喜欢长文阅读、在意排版细节的读者（书评/随笔/札记受众）
- **次要**：设计/前端圈里欣赏「反 AI 审美」的同行
- **非目标**：追热点、看速食内容的人

---

## 3. 内容策略

- 以读书札记为主体（重读经典、夜读抄、书评），占 70%
- 技术反思作辅（代码与工艺的随想），占 30%
- 纯文字，不配图
- 文章长度：中长文为主（5–15 分钟阅读）
- 更新频率：不定期，月刊或更慢

## 6. 内容模型（frontmatter 规范）

```yaml
---
title: 文章标题          # 必填，可用 *词* 标记斜体强调
deck: 一句话摘要          # 必填
date: 2026-07-02         # 必填，YYYY-MM-DD
read: 9                  # 必填，阅读分钟数
tags: [读书, 札记]        # 必填，数组
featured: false          # 可选，true 则上首页头条
series: reading-notes    # 可选，系列标识（同 series 的文章按 date 串联）
updated: 2026-07-10      # 可选，修订日期（显示「修订于」）
draft: false             # 可选，true 不构建
---
```

## 7. 设计规范

### 7.1 配色（oklch）

**日间（泛黄纸）**
| 令牌 | 值 | 用途 |
|---|---|---|
| `--paper` | `oklch(90% 0.04 85)` | 泛黄纸背景 |
| `--ink` | `oklch(25% 0.02 250)` | 墨黑（带青）正文 |
| `--accent` | `oklch(48% 0.09 200)` | 青灯青（主强调） |
| `--seal` | `oklch(50% 0.18 25)` | 朱砂红（辅助，印章/首字/引文/落款） |
| `--rule` | `oklch(80% 0.025 85)` | 分隔线 |

**夜间（墨色纸）**：paper `oklch(18% 0.01 250)` / accent `oklch(68% 0.1 200)` / seal `oklch(58% 0.16 25)`

### 7.2 字体

| 用途 | 字体 | 说明 |
|---|---|---|
| 展示（标题/刊名） | **Fraunces** + **思源宋体 Noto Serif SC** | 西文衬线 + 中文宋体 |
| 正文 | **Newsreader** + **思源宋体** | 可读衬线 |
| 元信息 | **IBM Plex Mono** | 日期/编号/标签 |

流体 `clamp()` 排版，完美四度 1.333，4pt 间距尺度。

### 7.3 版面

首页自上而下：① 报头 Masthead → ② 头条 Lead → ③ 主网格（编号文章索引 + 边栏：正在阅读/主题索引/订阅）→ ④ 关于作者 → ⑤ 版权页 Colophon。

### 7.4 交互

- 日夜切换、阅读进度、hover 朱砂短线滑入、入场动效
- 文章页：TOC 浮动、脚注返回、上下卷导航、返回顶部

### 7.5 响应式

移动优先，容器查询 `@container`，960px 主网格断点，`prefers-reduced-motion` 支持。
